## 🌐 Présentation

**Mini LinkedIn Académique** est une plateforme web inspirée des réseaux professionnels, dédiée au monde universitaire.  

Elle permet aux utilisateurs de :
- Créer un profil académique complet  
- Publier du contenu scientifique  
- Développer leur réseau  
- Collaborer sur des projets  

---

## 🚀 Fonctionnalités Principales

✔ Authentification sécurisée  
✔ Gestion de profil académique  
✔ Publication & interactions (likes, commentaires)  
✔ Réseau social (connexions)  
✔ Collaboration sur projets  
✔ Notifications en temps réel  
✔ Interface administrateur
